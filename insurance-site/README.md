# Kenan Aslan Sigorta - Insurance Company Website

A modern, responsive React website for Kenan Aslan Insurance Company, featuring a professional design with comprehensive insurance services information, contact forms, and appointment booking functionality.

## 🌟 Features

- **Responsive Design**: Fully responsive layout that works seamlessly across all devices
- **Modern UI**: Clean, professional design with Tailwind CSS
- **Contact Form**: Interactive contact form with validation
- **Appointment Booking**: Comprehensive appointment scheduling system
- **Google Maps Integration**: Location embedding for easy navigation
- **Service Portfolio**: Detailed insurance service offerings
- **Multi-page Navigation**: Smooth navigation between different sections

## 🛠️ Tech Stack

- **React 19.1.1**: Modern React with functional components and hooks
- **Vite**: Fast build tool and development server
- **Tailwind CSS 4.1.12**: Utility-first CSS framework for styling
- **React Router DOM 7.8.2**: Client-side routing
- **ESLint**: Code linting and quality assurance

## 🚀 Getting Started

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd insurance-site
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open your browser and navigate to `http://localhost:5173`
